# NeuroDetect AI

NeuroDetect AI is a Machine Learning and Deep Learning powered platform designed for the unified detection of neurodegenerative diseases such as Alzheimer’s and Parkinson’s. The system analyzes MRI scans, clinical records, and behavioral data using CNN-based models to support early diagnosis and accurate prediction.

## Features
- Alzheimer’s disease detection
- Parkinson’s disease detection
- MRI image analysis using Deep Learning
- Clinical and behavioral data processing
- Django-based web application
- Real-time prediction interface

## Technologies Used
- Python
- TensorFlow / Keras
- CNN, LeNet, SqueezeNet
- Django
- NumPy, Pandas, Matplotlib
- HTML, CSS, JavaScript
- SQLite3

## Purpose
The project aims to assist healthcare professionals by providing an AI-driven diagnostic support system for early and reliable detection of neurological disorders.
